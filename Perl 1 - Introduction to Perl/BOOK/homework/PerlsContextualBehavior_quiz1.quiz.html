Perl's Contextual Behavior - Quiz 1

Look at the documentation for the reverse() function with perldoc -f and construct two statements as follows:

  • Assume you have a scalar $phrase containing the string "A man, a plan, a canal--Panama," and assign to $rev the string that contains all the characters of that phrase in reverse order.
  • Assume you have an array @prices containing the prices of items in a store, ordered from least expensive to most expensive. Assign to $priciest the most expensive price in the array. (There are several ways of doing this. Choose a way that uses the reverse function in a single expression.)

my $rev = reverse "A man, a plan, a canal--Panama";

my ( $priciest ) = reverse @prices;

Grading Criteria: Student should demonstrate understanding of reverse in both scalar and list context.
