Perl's Contextual Behavior - Quiz 2

In the following code snippets, identify the context of EXPR (notation standing for whatever expression might be put at that place in the code).

Your choices are:

  • List
  • Scalar
  • Boolean (Scalar)

The snippets are:

  1. print EXPR
  2. if (EXPR) { print "Hello" }
  3. 17 + 42 * EXPR
  4. push @array, EXPR
  5. print reverse( "begin" .EXPR. "end" )
  6. while ( 0 + EXPR)
  7. foreach my $thing ( @array,EXPR)
  1. list
  2. boolean (scalar)
  3. scalar [defined by *]
  4. list
  5. scalar [defined by . (concat) Note these are not ,]
  6. scalar [defined by 0+ rather then while]
  7. list

Grading Criteria: Student should demonstrate understanding of correct context for a given expression in any situation.
