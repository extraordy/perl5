Strings and String Conditionals - Quiz 1

Explain why 1.0 == 1.00 but "1.0" ne "1.00".

== is a numeric comparison and 1.0 and 1.00 are the same numeric value.

eq is a string comparison and the strings do not have the same length, as one has an extra 0, thus cannot be considered the same.

Grading Criteria: Student should be able to explain the difference between these comparison operators.

If $name1 and $name2 contain names, how you would compare them while ignoring case?

Assume that they contain only characters from the basic ASCII character set.

Observe:
print 'same' if lc($name1) eq lc($name2);

-- or --

Observe:
print 'same' if uc($name1) eq uc($name2);

Grading Criteria: Student should be able to explain the use of either lc or uc in comparisons.
