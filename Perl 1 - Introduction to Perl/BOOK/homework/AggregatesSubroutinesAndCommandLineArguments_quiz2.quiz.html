Aggregates, Subroutines, and Command-Line Arguments - Quiz 2

Suppose you wrote a program called terraform.pl that takes command line options (arguments beginning with a hyphen). You might run the program this way:

Observe:
./terraform.pl -d --jovian "Phase 2 of Invasion Plan" Ganymede Callisto

How could you use index() to determine if a given argument was an option like -d or a "plain" argument like Ganymede?

Observe:
foreach(@ARGV) {
if(index($_, '-') == 0) { # use index to check to see if the first character is a -
print "$_ is an option\n";
}
else {
print "$_ is a plain argument\n";
}
}

Grading Criteria: Student should demonstrate understanding of how to parse @ARGV by using index().
