Properties - Quiz 1

Name one important difference between how Perl treats the number 0.0 and the string "0.0".

0.0 (is a number) will evaluate to false; "0.0" (is a string) will evaluate to true. This is the answer we're looking for.

Grading Criteria: Student should be able to explain the truth value for each version of 0.0.

The following code fragment is legal and produces the desired result but is confusing:

Observe:
unless ( $germ_count < 50 )
{
 print "Filthy\n";
}
elsif ( $germ_count > 25 )
{
 print "Dirty\n";
}
else
{
 print "Clean\n";
}

How is the code confusing?

How would you make it clearer?

Observe:
if( $germ_count >= 50) { ... }

Common Issues:

  • If the student overcomplicates their checks, for example by adding extra checks to elsif to check that $germ_count is < 50, instruct them that this check is not needed.
  • If the student inverts the order of Filthy/Dirty/Clean, instruct them that the only change that needs to be made is for the unless check and no other changes needs to be made.

Grading Criteria: Student should be able to describe the use of unless and double negatives and offer a solution using an if statement.
