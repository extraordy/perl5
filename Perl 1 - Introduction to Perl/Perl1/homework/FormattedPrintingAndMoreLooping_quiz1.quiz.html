Formatted Printing and More Looping - Quiz 1

What is the difference between the following code examples?

Observe:
while (my $x = shift @a )

and

Observe:
foreach my $x ( @a )

Hint: to test this, create and run a small program. After executing each loop, print the contents of the array.

The while example will likely result in an empty array when complete, because it removes values as it goes. The foreach example does not suffer this side effect.

Note: It is possible to still have values in @a if one of them is considered false, because the while will stop at the first false value, though this is often the undef at the 'end' of an array.

Often students will say that there is no difference between these two examples because both allow access to each value of the array individually. Instruct them that while this is true, there are side effects of the loop that are different.

Grading Criteria: Student should demonstrate understanding of loop constructs and shift.
