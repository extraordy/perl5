Interpolation - Quiz 1

Give a reason why you would use single quotes instead of double quotes to quote the terminating string of a heredoc (for example, <<'END_OF_TEXT').

  • Desire for a non-interpolated multi-line string.
  • If the string contains or might in the future contain dollar signs or at signs or backslashes that are to be rendered literally and that do not signify the beginning of variables or escape sequences to be interpolated or interpreted.
  • If the string contains or might in the future contain variables or escape sequences that should be interpolated or interpreted in the string rather than being rendered literally.

Grading Criteria: Student should be able to describe the use of single quotes.

Give a reason why you would use double quotes instead of single quotes to quote the terminating string of a heredoc (for example, <<"END_OF_TEXT").

  • Desire for an interpolated multi-line string.
  • If the string contains or might in the future contain variables or escape sequences that should be interpolated or interpreted in the string rather than being rendered literally.

Grading Criteria: Student should be able to describe the use of double quotes.
