Directory Reading Functions - Quiz 1

Name two differences between the results of the following two lines:

Observe:
opendir my $fh, "foo"; readdir($fh);

glob "foo/*"

readdir lists only filenames; glob leaves the path foo/" in each result.

readdir lists all the files.

glob will not include the files whose names begin with a period (in the example given).

Grading Criteria: Student should demonstrate knowledge of Perl directory functions.
