Grouping, Alternation, and Complete Parsing - Quiz 1

This regular expression for a zip code: /\d{5}(?:-\d{4})?/ matches the string "123456789-987654321". Why? How would you fix it?

The regex matches the part of the string "56789-9876".

To prevent that, use an anchor such as \b: /\b\d{5}(?:-\d{4})?\b/.

Grading Criteria: Student should demonstrate knowledge of using anchors in a regular expression.

What is $1 when the regular expression /(straw(?:berry|berrylike))/ is applied to "strawberrylike"? Why doesn't it match the whole string?

$1 is "strawberry" due to the test matching on 'berry' first and shortcutting.

Grading Criteria: Student should demonstrate knowledge of non-capture groups in a regular expression.
