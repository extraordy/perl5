Properties - Quiz 1

Experiment as necessary to answer the following questions:

1. What is the value of $_ following a foreach loop with no explicit loop variable?

2. Is the value of $_ changed following a call to a subroutine that modifies $_?

1. After a foreach loop, the value of $_ reverts back to what it was before the loop.

2. If a subroutine modifies $_, then the caller will see the change.

Grading Criteria: Student should demonstrate knowledge of how the default variable $_ is scoped and what effect should be expected because of this.

Which of the following functions can have an argument default to $_ (hint: perldoc -f):

  1. join
  2. split
  3. crypt
  4. pop
  5. chmod
  6. localtime
  7. kill

split is the only function in this list that uses the value of $_ if no values are passed to it.

Grading Criteria: Student should demonstrate knowledge of the default behaviors of a few common functions and the use of default values.
