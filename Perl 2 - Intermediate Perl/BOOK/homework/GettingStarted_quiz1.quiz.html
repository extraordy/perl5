Getting Started - Quiz 1

What is the difference between the exists() function and the defined() function with respect to testing hash elements?

exists() will return a true or false value based on if the given key exists in a hash, even if the value is undef.

defined() will return a true if a value is defined (not undef); otherwise it returns false.

Grading Criteria: Student should be able to explain both the exists() and defined() functions.

Given an array named @foo, how do you find out how many elements are in it?

Use @foo in a scalar context:

Observe:
$count = @foo;
