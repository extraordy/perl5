Regular Expressions: Quantifiers - Quiz 1

The regular expression /\d*\s*/ matches the string "WORD." Why? What part of that string is it matching?

It matches because it is capable of matching nothing, since it is composed entirely of atoms with quantifiers that are allowed to match zero items.

Grading Criteria: Student should demonstrate knowledge of some interesting side effects to using * as a quantifier in a regular expression.

When the regex /\w+\w*\w+/ is applied to the string freedom, what part does the first \w+ match, what part does the \w* match, and what part does the second \w+ match?

\w+ 'freedo'
\w* ''
\w+ 'm'

If the student is confused, explain that this is an example of backtracking.

Grading Criteria: Student should demonstrate knowledge of how a regular expression will often backtrack through a string to try and find a match.
