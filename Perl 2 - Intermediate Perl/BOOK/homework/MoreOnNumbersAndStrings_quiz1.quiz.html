More on Numbers and Strings; More on Regular Expressions; More on Hashes - Quiz 1

Explain the difference between /\b/ and "\b".

/\b/ is a regular expression fragment matching a word boundary (it would be useless as it stands without more).

"\b" is a string containing a backspace control character.

Grading Criteria: Student should be able to explain the difference between control characters and regular expression fragments.

You have the name of the author in $author and the path of this program in $0. What regex will match a string of the form "path was written by author"?

/\Q$0\E was written by \Q$author\E/. (The final \E may be omitted.)

Grading Criteria: Student should demonstrate knowledge of how to properly escape values in a regular expression.

How can you get a single quote inside a string in a one-liner? (Hint: perldoc perlop)

Use a code for the character; for example "\047" (ASCII code for single quote) or "\x27" (hex).

Grading Criteria: Student should demonstrate knowledge of how to use shell ASCII and hex character codes.
