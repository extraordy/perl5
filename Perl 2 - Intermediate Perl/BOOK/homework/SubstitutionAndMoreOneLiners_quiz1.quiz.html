Substitution and More One-Liners - Quiz 1

How can you get -i to prepend a prefix instead of append a suffix? (Hint: perldoc perlrun)

Put an asterisk (*) in the argument to -i and each applicable filename will be substituted for it, so you can put a prefix in front of the asterisk. If the student gives an example, make sure that the asterisk is quoted or escaped so that it is not interpreted by the shell for globbing.

Grading Criteria: Student should demonstrate knowledge of using the -i flag for Perl to make altered copies of a given file.

What does Perl do if -n or -p is specified and one or more files on the command line does not exist or cannot be read?

Prints a warning and carries on.

Grading Criteria: Student should demonstrate knowledge of how -n and -p relate to how Perl reads files via @ARGV.
