Regular Expressions: Character Classes - Quiz 1

If you didn't have the /i modifier, what regular expression would match "dog" regardless of case?

/[Dd][Oo][Gg]/

Grading Criteria: Student should demonstrate knowledge of character classes in a regular expression.

Give regular expressions to validate:

  1. A four-digit hexadecimal address in any case.
  2. A Canadian postal code (see specification in Wikipedia).
  1. /[0-9A-F][0-9A-F][0-9A-F][0-9A-F]/i (It is okay without slash delimiters if the character class is [])
  2. /[A-Z][0-9][A-Z] [0-9][A-Z][0-9]/ (Many bonus points if they read the spec completely and came up with [A-CEGHJ-NPRSTV-Z][0-9][A-CEGHJ-NPRSTV-Z][0-9][A-CEGHJ-NPRSTV-Z][0-9] or put the character class in a scalar first and used that in the regex.

Grading Criteria: Student should show they know how to use character classes.
