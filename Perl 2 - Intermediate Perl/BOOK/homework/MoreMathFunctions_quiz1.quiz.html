More Math Functions and Course Wrap-Up - Quiz 1

Read the first few pages of perldoc bignum. Write one-liners to find out how many extra digits of precision it provides for sqrt(0.5) on the student machine. (Hint: Read perldoc perlrun to see what flag to use to use a module from the command line.)

The answer depends on the student environment, but this dialog shows the process by which to get the answer:

Observe:
% perl -le 'print sqrt 0.5'
0.707106781186548
% perl -Mbignum -le 'print sqrt 0.5'
0.7071067811865475244008443621048490392848

The student may also write a program containing use bignum to get the second result. The answer from the above dialog is 42 - 17 = 25 extra digits.

Grading Criteria: Student should demonstrate knowledge of how to use bignum to increase Perl's floating-point precision.
