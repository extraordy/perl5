Introduction to One-Liners - Quiz 1

perl -le 'print 42' works; perl -el 'print 42' does not. Why?

Perl sees 'l' as the argument to the -e flag in the second case and executes that code. Due to the absence of the string and warnings pragma, no error is emitted, and the code does nothing.

Grading Criteria: Student should demonstrate knowledge of properly using arguments for running perl on the command line.

perl -le 'print 'Hello, Sailor!?'' does not work. Why? Which of the following Perl operators would let you get the intended effect:

  • qq
  • qw
  • qx
  • q
  • qr

(Hint: perldoc perlop)

The single quotes cause the shell to change what it interprets vs. what it passes to Perl. The q{} operator has the same effect as single quoting.

Grading Criteria: Student should demonstrate knowledge of when to use the appropriate quote-like operator.
